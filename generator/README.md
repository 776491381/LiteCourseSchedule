# Generator (生成器)

## 结构

主要是采用 JSON 结构，定义如下

```JS
[{
  "name": "课程名",
  "classroom": "教室信息",
  "start": 1, // 在第几节课开始，1 - 12
  "last": 0, // 持续几节课 1 - 12
  "day": 0 // 星期几的课，0-6 代表星期一到星期日
},
 // ...
]
```

只要生成的二维码按照上面的格式，课程就可以被正确导入。



## 正方教务系统

进入正方教务系统的选课系统，选择学生个人课表，此时打开浏览器的控制台（Console），将 [zhengfang.js](zhengfang.js) 复制进入即可产生课程二维码。扫码即可。
